######MEAL ME UP 

__Description:__
This is a website which will display finger licking delicious food 
recipies with proper ingridients and steps of how can you make yourself 
a yummy and mouth watering dish.


_Technologies involved_
AJAX
BOOTSTRAP
JAVASCRIPT 
CSS
HTML

__Endpoints__

####API with four **endpoints** are used to perform various tasks :

##ENDPOINT 1- TO LIST ALL MEALS 
...
"https://www.themealdb.com/api/json/v1/1/search.php?f=a"
...

##ENDPOINT 2- RANDOM MEALS 
...
"https://www.themealdb.com/api/json/v1/1/filter.php?i=chicken_breast"
...

##ENDPOINT 3- ALL MEAL CATEGORIES
...
"https://www.themealdb.com/api/json/v1/1/categories.php"
...

##ENDPOINT 4- INGRIDIENTS 
...
"https://www.themealdb.com/api/json/v1/1/list.php?i=list"
...
